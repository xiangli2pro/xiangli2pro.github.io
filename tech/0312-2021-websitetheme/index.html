Website Theme

Fri, Mar 12, 2021 2-minute read website


This is an ongoing post.

The website is powered by Hugo, employing the theme Anatole. It’s a beautiful template, and I would like to customize the setup a bit to reflect personal preference. I am a newcomer to the language css, js, basically I open the website in the chrome and use the developer tools to get a hint on which parameter changing which feature, a lot of functions are still under development. I will keep recording the changes I have made, in case I am going to modify it later or someone else wants a similar setup.

1. font (style.css)

  1. content font size: .p(16px);
  2. post title font size: .post .post-title(20px); .post .post-content(20px);
  3. post title not capitalized: add code below to .css file
h3 {
    text-transform: none !important;
}

 

2. layout (style.css)

  1. sidebar width: .content (70%), .page-top(70%); @media(30%), .sidebar(30%);
  2. category position (single.html): move category to the top of a post
<!--<div class="post-title">-->
     <!--<div class="info">-->
         <!--<em class="fas fa-calendar-day"></em>-->
         {{ with .Page.Params.Categories }}{{ partial "taxonomy/categories.html" . }}{{ end }}
    <!-- </div>
<!-- </div>

 

3. comment (.md)

  1. make utterance comment box show up in each post: need to set type= “post” in the .md file FrontMatter, and add “post” into mainSections in config.toml;

 

4. read more (index.html)

  1. add code below in the /anatole/layouts/index.html, between code chuck ‘post-content’ and ‘post-footer’
<!--  <div class="post-content"> <div>  -->

{{- if .Truncated -}}
  <a href='{{ .RelPermalink }}'>&hellip; Read more</a>
{{- end -}}

<!--  <div class="post-footer"> <div> -->

 

5. render .ipynb on Hugo

  1. install nbconvert
  2. create a text cell on top of the .ipynb file, add metadata (Front Matter) that you would normally setup for your hugo posts to the text cell.
  3. upload .ipynb file to your website …/content/posts/ folder
  4. go to command line under this folder path, render .ipynb to markdown by typing
jupyter nbconvert --to markdown <filename>.ipynb
  1. now you would see a <filename>.md file, you can make further changes
